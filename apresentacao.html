🌲 O que é uma Árvore N-ária?

📖 Definição

Uma árvore n-ária é uma estrutura hierárquica onde cada nó pode ter qualquer número de filhos (diferente de uma árvore binária que tem no máximo 2).


Ideal para representar:

  • 📁 Sistemas de arquivos
  • 🏷️ Categorias de produtos
  • 🌐 Organogramas
  • 📋 Menus hierárquicos

🔧 Operações

  • Inserir — Adicionar um nó filho
  • Buscar — Localizar um nó pelo nome
  • Remover — Excluir um nó e seus descendentes
  • Renomear — Alterar o nome de um nó
  • Contar — Total de nós na árvore
  • Profundidade — Maior nível hierárquico
  • Exportar — Gerar JSON da estrutura

📦 Classe Categoria

O nó fundamental da árvore n-ária

// Atributos public class Categoria { private String nome; private List<Categoria> subcategorias; // Inserir filho public void inserirSubcategoria(Categoria sub) { this.subcategorias.add(sub); } // Remover filho (recursivo) public boolean removerSubcategoria(String nome) { for (Categoria sub : subcategorias) { if (sub.nome.equalsIgnoreCase(nome)) { subcategorias.remove(sub); return true; } if (sub.removerSubcategoria(nome)) return true; } return false; } }

📌 Atributos

  • nome (String) — Identificador do nó
  • subcategorias (List) — Lista de filhos

🔍 Métodos Principais

  • inserirSubcategoria() — Adiciona filho
  • removerSubcategoria() — Remove recursivo
  • buscarPorNome() — Busca recursiva
  • contarNos() — Total de descendentes
  • profundidade() — Maior nível
  • toJSON() — Exporta formato JSON

📦 Classe ArvoreCategorias

Gerencia a árvore como um todo

public class ArvoreCategorias { private Categoria raiz; public ArvoreCategorias() { this.raiz = new Categoria("Loja"); } // Inserir no nivel principal public boolean inserirCategoriaPrincipal(String nome) { if (raiz.buscarPorNome(nome) != null) return false; raiz.inserirSubcategoria(new Categoria(nome)); return true; } // Inserir subcategoria em um pai especifico public boolean inserirSubcategoria(String pai, String nome) { if (raiz.buscarPorNome(nome) != null) return false; Categoria paiCat = raiz.buscarPorNome(pai); if (paiCat != null) { paiCat.inserirSubcategoria(new Categoria(nome)); return true; } return false; } // Remover (nao permite remover a raiz) public boolean removerCategoria(String nome) { if (raiz.getNome().equalsIgnoreCase(nome)) return false; return raiz.removerSubcategoria(nome); } // Renomear public boolean renomearCategoria(String antigo, String novo) { if (raiz.buscarPorNome(novo) != null) return false; Categoria cat = raiz.buscarPorNome(antigo); if (cat != null) { cat.setNome(novo); return true; } return false; } }

📌 Responsabilidades

  • Cria a raiz "Loja" no construtor
  • Valida duplicatas antes de inserir
  • Impede remocao da raiz
  • Fornece estatisticas (total, profundidade)

🔍 Fluxo de Insercao

1. Validar nome → 2. Buscar duplicata → 3. Localizar pai → 4. Inserir filho
