01 · Conceito

O que é uma Árvore N-ária?

📖 Definição

Uma árvore n-ária é uma estrutura hierárquica onde cada nó pode ter qualquer número de filhos (diferente de uma árvore binária, que tem no máximo 2).


Ideal para representar:

  • Sistemas de arquivos
  • Categorias de produtos
  • Organogramas
  • Menus hierárquicos

🔧 Operações (complexidade)

  • Inserir — adicionar um nó filho O(n)
  • Buscar — localizar um nó pelo nome O(n)
  • Remover — excluir um nó e seus descendentes O(n)
  • Renomear — alterar o nome de um nó O(n)
  • Contar — total de nós na árvore O(n)
  • Profundidade — maior nível hierárquico O(n)
  • Exportar — gerar JSON da estrutura O(n)

03 · O nó

Classe Categoria

O nó fundamental da árvore n-ária

// Atributos public class Categoria { private String nome; private List<Categoria> subcategorias; // Inserir filho public void inserirSubcategoria(Categoria sub) { this.subcategorias.add(sub); } // Remover filho (recursivo) public boolean removerSubcategoria(String nome) { for (Categoria sub : subcategorias) { if (sub.nome.equalsIgnoreCase(nome)) { subcategorias.remove(sub); return true; } if (sub.removerSubcategoria(nome)) return true; } return false; } }

📌 Atributos

  • nome (String) — identificador do nó
  • subcategorias (List<Categoria>) — lista de filhos, exposta como somente-leitura

🔍 Métodos Principais

  • inserirSubcategoria() — adiciona filho
  • removerSubcategoria() — remoção recursiva
  • buscarPorNome() — busca recursiva
  • contarNos() — total de descendentes
  • profundidade() — maior nível
  • toJSON() — exporta JSON com escape de caracteres

04 · A árvore

Classe ArvoreCategorias

Gerencia a árvore como um todo

public class ArvoreCategorias { private Categoria raiz; public ArvoreCategorias() { this.raiz = new Categoria("Loja"); } // Inserir no nível principal public boolean inserirCategoriaPrincipal(String nome) { if (nomeInvalido(nome) || raiz.buscarPorNome(nome.trim()) != null) return false; raiz.inserirSubcategoria(new Categoria(nome.trim())); return true; } // Inserir subcategoria em um pai específico public boolean inserirSubcategoria(String pai, String nome) { if (raiz.buscarPorNome(nome) != null) return false; Categoria paiCat = raiz.buscarPorNome(pai); if (paiCat != null) { paiCat.inserirSubcategoria(new Categoria(nome)); return true; } return false; } // Remover (não permite remover a raiz) public boolean removerCategoria(String nome) { if (raiz.getNome().equalsIgnoreCase(nome)) return false; return raiz.removerSubcategoria(nome); } // Renomear public boolean renomearCategoria(String antigo, String novo) { if (raiz.buscarPorNome(novo) != null) return false; Categoria cat = raiz.buscarPorNome(antigo); if (cat != null) { cat.setNome(novo); return true; } return false; } }

📌 Responsabilidades

  • Cria a raiz "Loja" no construtor
  • Valida nomes vazios e duplicatas antes de inserir
  • Impede remoção e renomeio da raiz
  • Fornece estatísticas (total, profundidade)

🔁 Fluxo de Inserção

1. Validar nome → 2. Buscar duplicata → 3. Localizar pai → 4. Inserir filho
